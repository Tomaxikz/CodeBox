This project is mainly for my school years work called CodeBox its inspired by perodactyl and calagopus panel
It is a app that allows schools to create classes or more admin assigns ownership of specific classes and those classes have max student count and assigned students by teachers. Students can then execute code in isolated docker containers therefore preventing possible damage to school hardware. Obviously teachers can view student container data, and admins can view everything it will use RBAC with assigneable and createable roles in the panel it self, students can also invite other students to containers across classes. Students can also create tenant databases for mysql/mariadb, postgresql, mongodb or redis. Students should be eable to execute JavaScript, TypeScript, PHP, Python, Rust, GoLang, Java, C#, Kotlin, C++, C, Ruby languages and use their respective package managers or runtimes choosen by the teachers assigning containers, User can interact with console, Files, Databases and so on of their assigned containers. Daemon will use Docker as container agent. 

This project was suppose to be written in BunJS and React for the panel and GoLang for the node daemon, but due to me needing to learn rust deeper and overall just leanr i choose to make panel Rust + React and Daemon Rust as well.
